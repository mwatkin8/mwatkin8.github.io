HGVS Tool

Convert an HGVS expression to a VR Allele

This tool will generate a VR Allele object to which represents the variant encoded by a user-entered HGVS expression. Included in this JSON representation of the Allele will be globally unique VR identifiers, as described in the Overview section of this tool suite.
